# Milestones

## Milestone 1: Privara Core Protocol + Testnet Router

Target: 4 weeks.

Success criteria:

- Clarity router contract deployed on Stacks testnet
- intent hash, nonce, and expiry system implemented
- relayer can execute authorized SIP-010 transfers
- tests cover replay protection and invalid authorization
- public GitHub repo and demo video of the working flow

## Milestone 2: SDK + Relayer + Developer Integration Layer

Target: 4 weeks after Milestone 1.

Success criteria:

- TypeScript SDK for intent creation and signing
- working relayer service handling testnet execution
- demo app showing end-to-end flow
- one external integration or simulated DAO payout flow
- developer documentation

## Final Adoption Metric

The first grant phase should produce at least one of:

- 25 successful testnet intent executions
- one external protocol or wallet integration demonstrating usage
